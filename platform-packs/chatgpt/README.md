# ChatGPT 安装包

这一套文件用于把 `Photo Pose Director` 配成：

- ChatGPT 自定义 GPT
- ChatGPT Custom Instructions 版本

## 目录说明

- `custom-gpt/`：给 ChatGPT 自定义 GPT 使用
- `instructions/`：给普通 ChatGPT 账号的 Custom Instructions 使用

## 推荐使用方式

如果你能创建自定义 GPT，优先使用 `custom-gpt/`。

如果你只是普通 ChatGPT 会话，使用 `instructions/` 里的两段内容填入 Custom Instructions 即可。
